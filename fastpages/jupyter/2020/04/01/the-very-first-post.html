Suporte para emoji no GitHub Pages

Com fast_template

Quando criei este blog os emoji não apareciam corretamente no GitHub Pages, para que apareçam tive de acrescentar o jemoji à lista de plugins no ficheiro _config.yml. Obrigado ao Arnie97 pela dica.

Exemplo de um _config.yml para fast_template:

plugins:
  - jekyll-feed
  - jekyll-gist
  - jekyll-octicons
  - jekyll-github-metadata
  - jemoji

Com fastpages

Entretanto fiz o upgrade para fastpages, e foi necessário:

  • adicionar o jemoji ao _config.yml
  • adicionar o jemoji ao Gemfile
  • make build
